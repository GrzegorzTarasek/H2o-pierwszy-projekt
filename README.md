Big Data Processing – H2O ML Project

Projekt zrealizowany w ramach przedmiotu Przetwarzanie Dużych Zbiorów Danych na studiach.
Celem projektu było wykorzystanie środowiska H2O do analizy oraz modelowania danych na większą skalę.

Cel projektu:

- Główne założenia projektu:
- praca z dużym zbiorem danych
- przygotowanie i transformacja danych
- budowa modeli uczenia maszynowego
- porównanie skuteczności algorytmów
- analiza jakości predykcji

Projekt miał na celu praktyczne zapoznanie się z narzędziami wykorzystywanymi w przetwarzaniu danych na większą skalę.

Wykorzystane technologie:
- Python
- H2O.ai
- Pandas
- NumPy
- Jupyter Notebook

Zakres realizacji

W projekcie wykonano:
eksploracyjną analizę danych (EDA)
- czyszczenie i przygotowanie danych
- podział na zbiór treningowy i testowy
- budowę modeli predykcyjnych w środowisku H2O
- ocenę modeli przy użyciu odpowiednich metryk
- analizę ważności zmiennych


Rezultaty:
- porównanie różnych algorytmów uczenia maszynowego
- identyfikacja najlepszego modelu na podstawie metryk jakości
- praktyczne zastosowanie narzędzi Big Data
- zdobycie doświadczenia w pracy z frameworkiem H2O
